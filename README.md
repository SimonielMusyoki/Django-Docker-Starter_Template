## Django Docker Starter Template
### <h3>Introduction</h3>
This is just a starter template to implement containerization for Django Apps, with both development and production environments

### <u>Tech Stacks</u>
Tecnologies Included:
1. Docker
2. Gunicorn
3. Nginx
4. Django(Python)
5. Postgress
